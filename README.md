# 📊 Sales Data Analysis Project

## 📌 Overview
This project analyzes sales data using Python and Pandas to generate insights such as total revenue, average sales, and best-selling product.

## 🛠️ Technologies Used
- Python
- Pandas

## 📊 Features
- Data cleaning (handling missing values, duplicates)
- Sales analysis (total, average, max, min)
- Best-selling product identification
- Automated report generation

## 🚀 How to Run
pip install pandas  
python sales_analysis.py  

## 📁 Files
- sales_analysis.py → Main code
- sales_data.csv → Input dataset
- cleaned_sales_data.csv → Cleaned dataset
- analysis_report.md → Output report

## 📈 Output
- Total Revenue
- Average Sales
- Highest & Lowest Sales
- Best Selling Product
